Recon isn't running

Start the daemon to see live headlines here every new tab.

  1. 1 Build the binary cd ~/path/to/recon/news-cli && go build -o recon ./cmd/recon && mv recon ~/.local/bin/
  2. 2 Install the systemd daemon recon daemon install
  3. 3 Check it's running recon daemon status
  4. 4 Open a new tab — you're live Ctrl + T
The daemon runs silently in the background, fetches every 15 min, and serves this extension at localhost:9645.
